My Web Pages

Web Page I learned ...
Famous Quotes How to code images and links within my documents. I also learned about many famous quotes attributable to important historical figures such as Gandhi.
Code of The West How to use inline style in my HTML documents. I also learned about the Code of the West, the once unwritten rules that people who used to live around here swore by.
Ready for Some Football How to use <table> elements and even more about inline style. I also learned about the generic block tag, <div>, and how it can be used to structure my web pages.
Fright Fest How to use embedded style by including a <style> tag in the <head> of my HTML document. I also learned how to use outside resources, link Google Fonts, and how to link them into my pages. My pages are looking a lot more refined now compared to the vanilla-looking ones we started with.
Play Ball How to go even further with embedded style. I learned about CSS selectors, in particular id and class selectors that allow you to apply CSS rulesets to individual page items or those that share something in common, respectively.
Rubik’s Cube Madness How to use the display property in my stylesheets to control the flow of my web pages. I also learned how to use the all-important float and clear properties to place elements on the page just where I want them. This knowledge will also help me create full-fledged websites with headers, footer, sidebars...you name it.
Rock, Paper, Scissors How to include JavaScript on a page and how it can be used to bring my web pages to life. I don’t understand all the JavaScript code just yet, but I do understand that it can interact with the elements that I place on the page. It is able to interact with the elements I place on the page using the same CSS selector rules I learned about earlier.
